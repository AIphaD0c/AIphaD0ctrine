# Lyrics

This page gathers the lyrical texts associated with **AIphaD0c**.

These works are presented as manifestations within **AIphaD0ctrine**.

## VIAL structure

- **VIAL.1**
- **VIAL.2**
- **VIAL.3**
- **VIAL.4**
- **VIAL.5**

Each VIAL contains its own internal theme, tone, and signal pattern.
